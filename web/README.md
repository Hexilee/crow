### Realtime 3D curve reconstruction ---- Frontend

#### Technology Stack

- TypeScript
- WebGL
    - three.js
- grpc-web

#### Development Tools(Recommended)

- yarn

```bash
> yarn install
```

#### Debug

```bash
> yarn dev
```

#### Release

```bash
> yarn build
```

Then you can serve `dist` fold.

```bash
> yarn serve
```

This script will run a simple `http-server` on `dist` fold.
